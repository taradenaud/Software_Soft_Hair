# Software Soft Hair Beauty Supply

A beauty supply and hair salon website featuring an interactive appointment booking system. Streamlined user experience for browsing services and scheduling appointments with specialized hairstylists.

## Features

- Service Catalog - Browse braiding services including Box Braids, Knotless Braids, Twists, Cornrows, and Stitch Braids
- Interactive Booking - Step-by-step appointment scheduling with real-time availability
- Stylist Selection - Choose from specialized braiders with detailed profiles
- Dynamic Pricing - Pricing calculator based on service type and hair length
- Payment Integration - Secure payment form with validation
- Responsive Design - Optimized for desktop and mobile devices
- Image Carousel - Showcase featured styles and promotions

## Tech Stack

- HTML5 - Structure and semantics
- CSS3 - Styling and animations
- JavaScript - Interactive functionality
- Bootstrap 4 - Responsive framework

## Getting Started

### Prerequisites

- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Python 3 (for local server) or Live Server extension

### Installation

1. Clone the repository
```bash
git clone https://github.com/taradenaud/Software_Soft_Hair.git
cd Software_Soft_Hair
```

2. Start a local server
```bash
python3 -m http.server 8000
```

3. Open [http://localhost:8000](http://localhost:8000) in your browser

### Alternative: VS Code Live Server

1. Install the "Live Server" extension in VS Code
2. Right-click `index.html` and select "Open with Live Server"

## Project Structure

- **index.html** - Main page with navigation and service showcase
- **app.js** - Appointment booking logic and form validation
- **styles.css** - Custom styling and layout
- **carousel_images/** - Hero carousel images
- **images/** - Service and UI images
